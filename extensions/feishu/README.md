# SupportClaw Feishu/Lark

Official SupportClaw channel plugin for Feishu and Lark workplace chats. Community maintained by @m1heng.

Install from SupportClaw:

```bash
supportClaw plugin add @supportclaw/feishu
```

Configure the Feishu/Lark app credentials in SupportClaw, then connect the plugin to the chats where agents should receive and send messages.
