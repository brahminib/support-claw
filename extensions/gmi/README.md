# SupportClaw GMI Cloud Provider

Official SupportClaw provider plugin for hosted GMI Cloud models through an
OpenAI-compatible API.

Install from SupportClaw:

```bash
supportClaw plugins install @supportclaw/gmi-provider
supportClaw gateway restart
```

Configure a GMI Cloud API key, then select models with refs such as
`gmi/google/gemini-3.1-flash-lite`.
