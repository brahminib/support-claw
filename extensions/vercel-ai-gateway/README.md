# Vercel AI Gateway SupportClaw provider

Official SupportClaw provider plugin for Vercel AI Gateway.

## Install

```sh
supportClaw plugins install @supportclaw/vercel-ai-gateway-provider
```

## Docs

See `docs/providers/vercel-ai-gateway.md` in the SupportClaw repository, or the published docs at `https://docs.supportClaw.ai/providers/vercel-ai-gateway`.
