# SupportClaw Codex

Official SupportClaw provider and harness plugin for OpenAI Codex app-server integration. It exposes the Codex-managed GPT model catalog and the Codex runtime surfaces used by SupportClaw agents.

Install from SupportClaw:

```bash
supportClaw plugin add @supportclaw/codex
```

Use this plugin when you want SupportClaw to run Codex-backed model turns, media understanding, and prompt overlays through the Codex app-server harness.
