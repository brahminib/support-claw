# SearXNG SupportClaw plugin

Official SupportClaw plugin for SearXNG.

## Install

```sh
supportClaw plugins install @supportclaw/searxng-plugin
```

## Docs

See `docs/tools/searxng-search.md` in the SupportClaw repository, or the published docs at `https://docs.supportClaw.ai/tools/searxng-search`.
